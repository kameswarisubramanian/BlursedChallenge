# Blursed Password App

Welcome to the Blursed Password App! This project is designed to provide the most frustrating password creation experience imaginable. Below are the details on how to set up and run the application.

## Project Structure

```
blursed-password-app
├── src
│   ├── index.html        # Main HTML document for the password creation interface
│   ├── app.js            # Main JavaScript file for handling user interactions
│   ├── styles.css        # Styles for the application
│   └── components
│       ├── passwordInput.js  # Manages the password input field
│       └── validator.js       # Validates the password against absurd rules
├── package.json          # Configuration file for npm
└── README.md             # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone [https://github.com/yourusername/blursed-password-app.git](https://github.com/kameswarisubramanian/BlursedChallenge.git)
   cd blursed-password-app
   ```

2. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application, open the `index.html` file in your web browser. The password creation interface will guide you through the absurd process of creating a password.

## Features

- **Dynamic Rules**: Password rules change while you type, ensuring you never know what to expect.
- **Contradictory Feedback**: The strength meter provides confusing and contradictory feedback on your password's strength.
- **Frustrating Confirmation Steps**: You will be required to confirm your password multiple times, with new rules introduced at each step.
